# 图书管理系统

## 项目简介

这是一个基于 C 语言实现的命令行图书管理系统，适合作为 C 语言课程项目。项目支持图书信息管理、借书还书、文件保存读取等功能。

## 功能列表

- 添加图书
- 显示所有图书
- 查询图书
- 删除图书
- 修改图书信息
- 借书
- 还书
- 文件保存与读取
- 预算购书推荐

## 技术栈

- C 语言
- 结构体
- 文件读写
- 函数模块化
- 基础数据处理

## 如何运行

```bash
gcc 图书管理系统.c -o library
./library
 ```

## 项目文件

```text
library/
├── README.md
└── 图书管理系统.c
```
